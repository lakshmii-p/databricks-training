# SQL Practice - Employees Table

## Objective
The objective of this practice session was to improve SQL query writing skills using different SQL concepts such as filtering, grouping, sorting, aggregation, logical operators, and pattern matching.

---

## Database Used
### Table: Employees

The `Employees` table contains employee-related information including:
- Employee ID
- Employee Name
- Department
- Salary
- City
- Experience

---

## Topics Covered

### SELECT Statements
- Retrieving all columns
- Retrieving specific columns
- Displaying employee details

### WHERE Clause
- Filtering data based on salary, city, department, and experience

### GROUP BY
- Department-wise and city-wise aggregation
- Counting employees
- Calculating total and average salaries

### HAVING Clause
- Filtering grouped data using conditions

### TOP / LIMIT
- Retrieving highest paid employees
- Finding employees with highest experience

### DISTINCT Keyword
- Displaying unique values from columns

### Comparison Operators
- Using `>`, `<`, `>=`, `<=`, `<>`

### Logical Operators
- Using `AND`, `OR`, and `NOT`

### IN and NOT IN
- Filtering multiple values efficiently

### BETWEEN Operator
- Selecting values within ranges

### LIKE Operator
- Pattern matching using `%` wildcard
